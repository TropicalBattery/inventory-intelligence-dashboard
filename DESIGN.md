# Inventory Intelligence Design System

Typography conventions (Tailwind):

| Role | Classes |
| --- | --- |
| Page title | `text-2xl font-semibold text-slate-900` |
| Section header | `text-lg font-semibold text-slate-900` |
| Stat label | `text-sm font-medium uppercase tracking-wide text-slate-500` |
| Body | `text-sm text-slate-700` |
| Table header | `text-xs font-medium uppercase text-slate-500` |

Color palette:

| Token | Usage |
| --- | --- |
| `bg-slate-50` | App background |
| White + `border-slate-200` + `shadow-sm` | Cards |
| `bg-slate-900` | Sidebar |
| `bg-indigo-600` | Primary accent (buttons, links, active marks) |
| Badge variants | See `components/ui/Badge.tsx` |

Spacing: page content uses `px-8 py-8`; cards use `p-6` by default.
